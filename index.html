Welcome!

I created this website after my Pluslife order was one of multiple denied entry into Australia at the border due to lack of an import permit - a requirement I was unaware existed. Following this, I have gathered a lot of information from Australian Government Department of Agriculture, Fisheries and Forestry about how to legally and successfully import Pluslife tests into the country.

My goal for this website is to collect all the information I have gathered about importing Pluslife tests into Australia into one place, such that legally importing these tests into Australia is as accessible to as many people as possible.

Website Guide

This website includes the following pages:

  • Overview: a brief summary of the legal status of ordering Pluslife tests to Australia, which acts as a little tl;dr for the whole website. Start here.

  • FAQ: an FAQ section outlining in detail the current legal status of ordering pluslife tests to Australia without a permit, what will happen if any in-transit order gets stopped at the border without a permit, and various other import-related queries you may have.

  • Applying for Import Permits: a how-to guide explaining how to apply for a Pluslife import permit.

  • Using Import Permits: a how-to guide explaining how to use an import permit to order Pluslife tests.
